# Infinity Jump

파이썬으로 만든 플랫포머 게임

조작 : Space bar

디노 게임을 오마주한 게임으로 

기본적인 화면은 스코어, 노랑색 플레이어, 장애물이다. 
이 게임은 장애물을 피하고 점수를 쌓는 게임 입니다.

만약 장애물에 닿으면 게임 오버되고 다시 r키를 누르면 다시 시작된다.
